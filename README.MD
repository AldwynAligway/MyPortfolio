# Personal Developer Portfolio 🚀

A lightweight, high-performance personal developer portfolio built with a vanilla frontend and integrated with a headless CMS to dynamically showcase software projects, desktop GUI applications, and system design work.

🌐 **Live Website:** [Insert Your Live Link Here]

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Headless CMS:** Sanity CMS (API integration for dynamic project management)
* **Version Control & Hosting:** GitHub, Vercel / Netlify

---

## 🔑 Key Features

* **Dynamic Project Fetching:** Integrates directly with the Sanity Content Lake API to render project cards, descriptions, tags, and links dynamically.
* **Zero Framework Bloat:** Built with raw HTML, CSS, and JS for near-instant load times and high performance.
* **Fully Responsive:** Custom flexbox and grid layouts optimized across desktop, tablet, and mobile displays.
* **Project Showcase:** Features full-stack web platforms, data tools, and desktop applications built with Python/PyQt5.
